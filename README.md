# FenricHomebrew
D&D 5e özel içerik koleksiyonu Aurora Builder için

## İçerik

### Irklar
- **Harengon** - Tavşan benzeri yaratıklar

### Alt Sınıflar
- **Chronurgy Magic** - Zaman büyülerinde uzmanlaşmış Wizard alt sınıfı

### Geçmişler
- **Scholar** - Akademik ve araştırmacı karakter geçmişi

### Yetenekler
- **Alert** - İnisiyatif ve algı bonusları

### Büyüler (Chronomancy)
#### Cantrips (0. seviye)
1. **Fast-Forward/Reverse** - Nesneleri hızlandırma/geri sarma
2. **Undo Harm** - Taze yaraları iyileştirme

#### 1. Seviye
3. **Déjà Vu** - Önceki aksiyonu tekrar ettirme
4. **Gift of Alacrity** - İnisiyatif bonusu
5. **Halt** - Hareket durdurma (reaction)

#### 2. Seviye
6. **Grist** - Nesneleri hammaddeye çevirme
7. **Hurry/Delay** - İnisiyatif manipülasyonu
8. **Recall** - Önceki pozisyona ışınlanma

#### 3. Seviye
9. **Haste** - Gelişmiş hızlanma
10. **Slow** - Çoklu hedef yavaşlatma
11. **Time's Arrow** - Mermileri durdurma ve yönlendirme (reaction)

#### 4. Seviye
12. **Dire Warning** - Gelecekten mesaj alma
